Данный скрипт парсит лог после прогона скриптов на БД, полученный sqlplus spool`ом
В результирующий файл попадают только уникальные ошибки + фиксируется время выполнения каждого скрипта

Установка и эксплуатация: 
	1. Данный скрипт запускается .bat файлом с параметрами в процессе установки (сразу после закрытия spool`а)
	2. Параметры:
	   - Путь до директории с логами
	   - Имя файла "*.log", который парсим

Примечание: 
Пример батника: work_project/Parsing_logs/pars.bat
